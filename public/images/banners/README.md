# Banner Images

Add your banner images here with the following names:

1. **banner-1.jpg** - Personalized Gifts banner (1600x600px recommended)
2. **banner-2.jpg** - Custom Photo Frames banner (1600x600px recommended)
3. **banner-3.jpg** - Gift Hampers banner (1600x600px recommended)

## Recommended Specifications:
- **Size**: 1600 x 600 pixels (or similar aspect ratio)
- **Format**: JPG or PNG
- **Quality**: High quality, optimized for web

## Tips:
- Use images that showcase your actual products
- Keep important content centered (for mobile crop)
- Avoid text in images (text is overlaid via code)
